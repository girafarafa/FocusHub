# 🎯 FocusHub

**FocusHub** este o aplicație modernă și intuitivă concepută pentru a-ți spori productivitatea și a te ajuta să-ți gestionezi mai bine timpul și sarcinile zilnice.

---

## 🌐 Demo Live

Aplicația poate fi testată direct în browser accesând link-ul de mai jos:

👉 **[Deschide FocusHub Live](https://girafarafa.github.io/FocusHub/)**

---

## 🚀 Caracteristici

* ⏱️ **Timer de Productivitate:** Conceput pentru sesiuni de lucru concentrate și pauze eficiente.
* 📋 **Managementul Sarcinilor:** Adăugarea și organizarea rapidă a activităților zilnice.
* 🎨 **Interfață Modernă:** Design curat, intuitiv și ușor de utilizat.
* 🖥️ **Cross-Platform:** Funcționează atât în browser, cât și ca aplicație desktop (Electron).

---

## 🛠️ Tehnologii Utilizate

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Desktop Runtime:** [Electron](https://www.electronjs.org/)
* **Version Control:** Git & GitHub
